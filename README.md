Nome do Projeto: LeiloesTDSat

Sobre o Projeto:
O LeiloesTDSat e um sistema de gerenciamento de leiloes desenvolvido em Java.
O sistema permite cadastrar, listar, editar e gerenciar produtos disponiveis
para leilao, controlando informacoes como nome, valor e status de venda dos itens.

Tecnologias Utilizadas:
- Java
- MySQL
- Java Swing
- JDBC
- Apache NetBeans
